## Summary

Following things we have covered in this session:

* Basic Routing
* Children Routes
* Route Guards (`CanActivate`, `CanActivateChild`, `Resolvers`, etc)
* Lazy Loading Modules
* Preloading Modules
* Custom Preloading Strategy
